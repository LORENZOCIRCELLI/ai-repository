# AI Repository

This repository contains various AI projects and resources. It is a collection of code, models, and documentation related to artificial intelligence. The projects range from machine learning algorithms to natural language processing and computer vision. It's main focus is to contain all the studies related to these fields and serve as a only knowledge acquirement repository.

---

## Machine Learning Libraries

- PyTorch
- Scikit-learn
- Tensorflow
- Keras

---

Study Objectives:

- Understand the fundamentals of machine learning algorithms and their applications.
- Comprehend the mathmatical foundations of machine learning, including linear algebra, calculus, and probability.
- Not only write machine learning code, but also understand the underlying principles and concepts.
- Be able to apply machine learning techniques to real-world problems and datasets.
